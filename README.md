我设想了一个双通道多色3D打印机头方案，当3D打印机工作时双通道系统使得一通道内的耗材始终属于待机状态，而另一个通道内的耗材在滑动。此时由于双通道的存在，则不需要像单通道那样进行长距离的缩回耗材动作， 以此来缩短换色过程中抽取和送料的时间来达到增加多色3D打印的工作速度。这个方案目前已经接近完成，我希望能有更多的参与者来一起挖掘这个方案的可能性
